Optionally connect an Azure AI Speech resource for higher-quality voices and phoneme-level pronunciation scoring. Everything works on the free browser engine without this.

Use Azure Text-to-Speech Neural voice playback instead of the browser's built-in voice.
Use Azure Pronunciation Assessment Phoneme-level accuracy/fluency scoring instead of simple word matching.
Azure Speech credentials
🔒 Your key is stored only in this browser's local storage and sent directly from your browser to Azure — it never passes through any server of ours. Because this app has no backend, the key is visible in this browser's dev tools/network requests, so only use a key you're comfortable having exposed on this device (fine for personal use or a small trusted pilot; not safe for a public unauthenticated deployment). See README for how to create a free-tier Azure AI Speech resource.
Text-to-speech cache Synthesized audio is cached on this device so replaying the same line doesn't call the Azure API again.
